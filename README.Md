# Sistema de monitorización de una planta:

## Componentes:

### Nodo 1:

- Sensor fotovoltaico
- Sensor de humedad/temperatura
- Sensor de humedad enterrado (diferente al otro)
 
### Nodo 2:

- LCD con mensajes concretos
- LED con semaforo para saber si todo OK o no


### Servidor

- Monitorización y graficas en tiempo real de los parametros de cada planta (de haber mas de una)
